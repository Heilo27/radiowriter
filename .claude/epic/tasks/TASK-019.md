**Issue:**
Interactive buttons use only icons without accessibility labels. VoiceOver users hear "button" with no context.

**Fix:**
```swift
Button {
    showingAddContact = true
} label: {
    Image(systemName: "plus")
}
.accessibilityLabel("Add contact")
```

Apply to ALL icon-only buttons throughout the app.
