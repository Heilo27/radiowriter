## Description

**Source:** Code Review 2026-02-02 (Lumen)
**Severity:** Critical
**Category:** Accessibility

**Location:** ContentView.swift:142-163

**Issue:**
Toolbar buttons (Read, Write, Clone) lack keyboard shortcuts. This is a macOS HIG violation - primary actions should have Command+key shortcuts.

**Recommendation:**
```swift
Button("Read", systemImage: "antenna.radiowaves.left.and.right") {
    Task { await coordinator.readFromRadio() }
}
.keyboardShortcut("r", modifiers: [.command])

Button("Write", systemImage: "antenna.radiowaves.left.and.right.slash") {
    Task { await coordinator.writeToRadio() }
}
.keyboardShortcut("w", modifiers: [.command, .shift])
```

Also add shortcuts to zone/channel operations in ZoneChannelView.
