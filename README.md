# RadioWriter

**Motorola radio programming software for Mac.**

Finally, program your Motorola radios without booting into Windows. Set frequencies, channels, tones, and settings—all from your Mac.

## Download

**[Download Latest Release](https://github.com/Heilo27/radiowriter/releases/latest)**

Requires macOS 12 or later. Universal binary for Intel and Apple Silicon.

## Features

- **Channel Programming** — Set up channels with frequencies, names, and power levels
- **Frequency Management** — Configure TX/RX frequencies, offsets, and splits
- **Tone & Squelch** — CTCSS, DCS, and digital squelch codes
- **Codeplug Support** — Read, write, and edit codeplug files
- **Native Mac App** — No emulation, no Wine, no virtual machines
- **Open Source** — Full source code available

## Supported Radios

- CP200d
- CP185
- PR400
- PM400
- GM300
- GM350
- More coming...

Don't see your radio? [Open an issue](https://github.com/Heilo27/radiowriter/issues) and we'll look into adding support.

## Requirements

- macOS 12 (Monterey) or later
- Motorola-compatible USB programming cable
- A Motorola radio to program

## Installation

1. Download the latest `.dmg` from [Releases](https://github.com/Heilo27/radiowriter/releases)
2. Open the DMG and drag RadioWriter to Applications
3. Connect your programming cable and radio
4. Launch RadioWriter and start programming
